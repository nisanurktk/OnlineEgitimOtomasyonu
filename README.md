Bu proje, bir online eğitim platformunun temel işlevlerini yönetmek amacıyla Java programlama dili kullanılarak geliştirilmiştir.
Sistem; öğrenci, öğretmen, ders ve not yönetimi gibi temel modülleri kapsamaktadır.
Projenin temel özellikleri arasında yeni kayıt oluşturma, mevcut bilgileri güncelleme,haftalık ders programlarını düzenleme, ders içeriklerini görme ve öğrencilerin başarı notlarını sisteme girme gibi fonksiyonlar yer almaktadır. 
Uygulamanın arayüzü Java Swing ile hazırlanmış olup, tüm veriler MySQL veritabanı ile güvenli bir şekilde saklanmaktadır.
